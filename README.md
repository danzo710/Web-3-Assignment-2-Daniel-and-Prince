- **Home View:**  
  A simple landing page with a banner introducing the site.

- **Browse View:**  
  Allows users to filter products by category, size, color, and gender, and supports sorting options.

- **Product Details View:**  
  Users can select size, color, and quantity, then add items to the cart.

- **Cart View:**  
  Displays selected products with options to add or remove items, along with subtotal, tax, shipping, and total calculations.

- **Gender View:**  
  Shows product categories for the selected gender with quick navigation buttons leading to the Browse view filtered for that gender.

- **Login View:**  
  Allows an admin to log into the site. Admin login currently uses no full authentication system.

- **Admin Product View:**  
  A popup inside the Product Details page that displays sales data for admins.

- **Sales Dashboard:**  
  Simple dashboard showing the sales data for all products such as Top seelling products and categories. Only avaulable to admins.
